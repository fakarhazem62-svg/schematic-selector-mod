# Schematic Selector Mod
**Minecraft 1.21.1 | Fabric**

## كيف تشتخدمه

1. **شغّل اللعبة** مع الفابريك
2. **روح على** Options → Controls → Schematic Selector
3. **ضبط الأزرار** اللي تريدها (الافتراضي: V للتحديد، M للحفظ)

## طريقة الاستخدام

| الزر | الوظيفة |
|------|---------|
| **V** | تفعيل/إلغاء وضع التحديد |
| **زر الضرب (بالفأرة اليسار)** | تحديد الزاوية الأولى (Pos1) |
| **زر البناء (بالفأرة اليمين)** | تحديد الزاوية الثانية (Pos2) |
| **M** | فتح شاشة الحفظ |

## خطوات الحفظ

1. اضغط **V** — يظهر لك رسالة "Selection Mode: ON"
2. اضغط **زر الضرب** على البلوكة الأولى
3. اضغط **زر البناء** على البلوكة الثانية
4. اضغط **M** — تفتح شاشة الحفظ
5. اكتب اسم الملف وادغط **Save**
6. الملف يُحفظ في: `.minecraft/schematics/<اسمك>.nbt`

## ملف الـ Schematic

- الملفات تُحفظ بصيغة **NBT** (`.nbt`)
- مجلد الحفظ: `.minecraft/schematics/`
- متوافق مع Structure Blocks وأغلب أدوات البناء

## Build

```bash
./gradlew build
```

الـ JAR يكون في: `build/libs/schematic-selector-1.0.0.jar`
